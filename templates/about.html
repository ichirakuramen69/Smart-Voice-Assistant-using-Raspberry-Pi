Project Overview

Raspberry Pi Voice Assistant with Smart Home Controls

About the Project

This voice assistant is a comprehensive IoT solution built on Raspberry Pi, featuring advanced speech recognition, natural language processing, and hardware control capabilities. The system responds to voice commands to control LEDs, servo motors, and can engage in intelligent conversations using Google's Gemini AI.

The project demonstrates the integration of multiple technologies including offline speech recognition, text-to-speech synthesis, hardware control through GPIO, and cloud-based AI services.

Key Features

Wake Word Detection

Responds to "assistant" wake word using Vosk speech recognition

LED Control

Turn on/off lights and create blinking patterns via voice commands

Servo Control

Control fan speed with voice commands and PWM control

AI Integration

Powered by Google Gemini for intelligent conversation responses

Voice Recording

Record and save voice interactions with timestamp logging

Text-to-Speech

Natural voice responses using Rhasspy TTS engine

Technology Stack

Python 3 Raspberry Pi 4 Vosk Speech Recognition Google Gemini AI Rhasspy TTS Flask Web Framework Socket.IO GPIO Zero SoundDevice

System Architecture

1
Audio Capture: Continuous audio stream capture using SoundDevice with 48kHz sampling rate
2
Wake Word Detection: Vosk processes audio stream to detect "assistant" wake word
3
Command Processing: Rhasspy NLU analyzes voice commands and extracts intents
4
Action Execution: System executes hardware controls or queries Gemini AI
5
Response Generation: TTS engine provides audio feedback to user

Hardware Components

Raspberry Pi 4: Main processing unit running the voice assistant software

LEDs: Connected to GPIO pins 17 and 26 for visual feedback and lighting control

Servo Motor: Connected to GPIO pin 18 with PWM control for fan speed management

USB Microphone: High-quality audio input for voice command capture

Speaker/Audio Output: For TTS voice responses and system feedback

Voice Commands

The system recognizes various voice commands including:

  • "Turn on/off light" - Controls LED state
  • "Blink light" - Creates blinking LED pattern
  • "Turn on/off fan" - Controls servo motor
  • "Increase/decrease speed" - Adjusts fan speed
  • "Enable/disable recording mode" - Controls voice logging
  • General conversation - Handled by Gemini AI
